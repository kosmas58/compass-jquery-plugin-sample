jQuery Media Helper Utilities

Some small (2kb) jQuery utilities for cross-browser responsive design, testing CSS media queries, cross-browser orientationchange event support, and more.

All code is pulled from jQuery Mobile source. Check out the README on Github or the source code for examples.
