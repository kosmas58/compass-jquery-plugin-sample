Thanks to Mark Williams we have another great feature - Column Reordering.
This is done again just with setting one options - sortable. Click on header column and try to move it to another location.
This feature depend on jQuery UI sortable widget.
Since we have other great stuffs which are depend on jQuery UI we have created a separated module
grid.jqueryui.js

HTML <table id="colr"></table> <div id="pcolr"></div> Java Scrpt code jQuery("#colr").jqGrid({ sortable: true, url:'server.php?q=2', datatype: "json", colNames:['Inv No','Date', 'Client', 'Amount','Tax','Total','Notes'], colModel:[ {name:'id',index:'id', width:55}, {name:'invdate',index:'invdate', width:90}, {name:'name',index:'name asc, invdate', width:100}, {name:'amount',index:'amount', width:80, align:"right"}, {name:'tax',index:'tax', width:80, align:"right"}, {name:'total',index:'total', width:80,align:"right"}, {name:'note',index:'note', width:150, sortable:false} ], rowNum:10, rowList:[10,20,30], pager: '#pcolr', sortname: 'invdate', viewrecords: true, sortorder: "desc", caption:"Column Reordering Example" }); jQuery("#colr").jqGrid('navGrid','#pcolr',{add:false,edit:false,del:false});
